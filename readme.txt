== Description ==

Automatr updates your WordPress automatically, never again do you have to worry about applying updates to WordPress. It supports updating plugins, themes and WordPress itself. 

Your modifications are fine with this plugin, as you can exclude them from the updating proccess easily.

== Installation ==

1. Install Automatr either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin.
3. Enable the updates that you would like to be applied automatically via the options menu.

== Frequently asked questions ==

= Will this delete any customisations I have made myself? =

If you have made some customisations to a plugin or theme, then simply exclude it from the update list (found in the settings page)

= How often will this check for updates? =

Currently the update is scheduled to run every hour.
